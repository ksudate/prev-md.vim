# prev-md.vim

Vim plugin for markdown preview

## Installation

To install using [Vim-Plug](https://github.com/junegunn/vim-plug):

```
" add this line to your .vimrc
Plug 'tmrekk121/prev-md.vim'
```

## Requirements

- [MichaelMure/mdr](https://github.com/MichaelMure/mdr)

## Usage

```
:Prevmd
```
